# HYROX Planner

Um aplicativo em Python para gerenciar e organizar treinos de HYROX de forma simples e intuitiva.

## 📋 Descrição

O HYROX Planner é um programa de linha de comando que permite:
- ✅ Adicionar treinos padrão ou exercícios específicos
- 📊 Visualizar todos os treinos cadastrados
- ✏️ Editar treinos existentes
- 🗑️ Excluir treinos

## 🚀 Funcionalidades

### 1. Adicionar Treinos
- **Treino Padrão**: Registre nome, duração, horário e intensidade
- **Exercício Específico**: Registre exercícios do HYROX com tempo, distância, carga e repetições

### 2. Visualizar Treinos
- Tabelas formatadas mostrando todos os treinos padrão
- Lista de exercícios específicos com detalhes completos

### 3. Editar Treinos
- Modifique informações de treinos padrão
- Altere dados de exercícios específicos

### 4. Excluir Treinos
- Remova treinos padrão
- Delete exercícios específicos

## 💾 Armazenamento de Dados

Os dados são salvos automaticamente em um arquivo `hyroox_planner_dados.txt` em formato de texto simples, permitindo fácil acesso e portabilidade.

## 🛠️ Como Usar

1. Execute o programa:
   ```bash
   python hyrox_planner.py
   ```

2. Escolha uma opção no menu:
   - `1` - Adicionar treinos
   - `2` - Visualizar treinos
   - `3` - Editar treinos
   - `4` - Excluir treinos
   - `0` - Sair

## 📝 Requisitos

- Python 3.x
- Sistema operacional com suporte a comando `cls` (Windows) ou adaptação para Linux/Mac

## 🎯 Exemplos de Uso

- **Treino Padrão**: Corrida matinal com 45 minutos de duração, intensidade moderada
- **Exercício Específico**: Sled Push com 80kg, 100m de distância, 30 repetições
